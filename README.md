# 🍽️ Isha Restaurant

A modern and responsive restaurant website built with **Next.js**, **TypeScript**, and **Tailwind CSS**. The platform allows customers to explore the restaurant menu, add dishes to a cart, and place orders directly through **WhatsApp** for a seamless ordering experience.

---

## 🌟 Features

* 🍽️ Beautiful and responsive restaurant landing page
* 📋 Categorized digital menu
* ➕ Add/Remove quantity for each menu item
* 🛒 Interactive shopping cart
* 📱 WhatsApp order integration
* 📍 Google Maps location
* 🚚 Dine-In, Takeaway & Home Delivery support
* ⚡ Fast and responsive UI
* 📱 Mobile-friendly design
* 🎨 Clean and modern interface

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Jadav-Ruthwik/isha-restaurant.git
```

Move into the project

```bash
cd isha-restaurant
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```text
http://localhost:3000
```

---

## 📱 WhatsApp Ordering Flow

1. Browse the menu
2. Select dish quantities
3. Review cart
4. Click **Order on WhatsApp**
5. Restaurant receives the formatted order
6. Restaurant confirms the order and shares the location if required
